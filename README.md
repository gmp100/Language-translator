A simple Language Translator Web App built with React + Vite and styled using Tailwind CSS. It uses the MyMemory Translation API to translate text between different languages.

Features

Translate text between multiple languages

Fast and responsive UI

Clean design with Tailwind CSS

Real-time translation using API

Tech Stack

React.js

Vite

Tailwind CSS

MyMemory Translation API

Installation
# Clone repository
git clone https://github.com/your-username/language-translator.git

# Navigate into project folder
cd language-translator

# Install dependencies
npm install

# Start development server
npm run dev
